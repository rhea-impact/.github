## Free Software for Coding Agents

Alongside our humanoid-robot projects, we release **free, open-source software** designed to work with coding agents and AI copilots.

Our goals:
- Turn everyday life tasks into repeatable agent workflows
- Make productivity tools accessible to people who aren't programmers
- Open up patterns and reference implementations anyone can adopt or improve

### Projects

| Repo | What it does |
|------|-------------|
| [`space-hog`](https://github.com/rhea-impact/space-hog) | CLI tool for finding and reclaiming wasted disk space on macOS. Prioritized cleanup advice, Docker analysis, AI-friendly recommendations. |
| [`applaud`](https://github.com/rhea-impact/applaud) | A chat-native Claude Code client with hot-swappable themes, built with Svelte and Tauri. |

## How You Can Use Our Software

- As building blocks for your own coding-agent setups
- As reference implementations for agent-safe patterns and UX
- As educational examples for students, nonprofits, and community labs

We welcome issues, discussions, and pull requests from anyone building agents that help people in their day-to-day lives.
